# ClinicDesk

ClinicDesk is a PHP and MySQL clinic management dashboard with role-based access for admins, doctors, and patients.

## Requirements

- PHP 8+
- MySQL / MariaDB
- WAMP, XAMPP, Laragon, or similar local server
- Apache
- AdminLTE 3
